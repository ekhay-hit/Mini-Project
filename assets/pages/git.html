What is a git branch

A branch represents an independent line of development.It allow us to develop features, fix bugs, or safely experiment with new ideas in a contained area of your repository <2> Step of creating a branch
    Creat a new branch named branch-Name
  • Git checkout -b feature/branch-Name
    • gives list of branch and astrike gives the current branch
  • git branch
    • Opened the branch in VsCode
  • code .
    • create a JavaScript file named index
  • touch index.js
    • Added the file to be ready for a commit
  • git add -A
    • Commit the file the changes to GitHub
  • git commit -m ”created index.js”
    • exit the current branch an open main branch
  • git checkout main
    • merge the the feature/branch-Name with main branch
  • git merge /feature/branch-Name
    • Delete the feature/branch-Name
  • git branch -d feature/branch-Name
